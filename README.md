# Homebrew Tap for Cangjie ecosystem

This tap provides an easy way to install components of the Cangjie programming ecosystem on macOS (Linux support coming soon) via Homebrew.

It compiles and installs the following projects:

- [Cangjie compiler](https://gitcode.com/Cangjie/cangjie_compiler)
- [Cangjie runtime and standard library](https://gitcode.com/Cangjie/cangjie_runtime)
- [Cangjie stdx](https://gitcode.com/Cangjie/cangjie_stdx)

## Installation

First, install Homebrew by following the instructions in the [official documentation](https://docs.brew.sh/Installation).

Then tap the repository and install the cangjie formula:

```
brew tap cangjie/core https://github.com/CJPLUK/homebrew-cangjie.git
brew install cangjie
```
